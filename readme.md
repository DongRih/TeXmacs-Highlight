# TeXmacs-Highlight

A TeXmacs package providing highlight functionality.

## Installing

1. Copy the `.ts` file into:
	- (Windows) `%APPDATA%\TeXmacs\packages`
	- (Linux) `~\.TeXmacs\packages`
2. Include this package in your document.
